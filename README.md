#Brasilcessível

Este site representa o MVP de um produto de acessibilidade para eventos culturais brasileiros.

A plataforma visa mostrar os eventos culturais (exposições, peças de teatro, filmes, etc.) com suas classificações de acessibilidade para consulta de interessados nelas.

A base de dados, por enquanto, é apenas exemplificativa do formato dos dados e não reflete a realidade (este é um MVP!!!) e está feita no [Google Sheets](https://docs.google.com/spreadsheets/d/15m2m4X90XtI5ycZ58wu5xUz5_NTXp82_CJuK_ntkd5U/edit?usp=sharing)
